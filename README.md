# Srivibhu Piratla Engineering Portfolio

Static engineering portfolio site built for GitHub Pages.

## Files

- `index.html` - main portfolio page
- `styles.css` - responsive styling
- `script.js` - click-to-open project detail interactions
- `assets/project-images/` - images extracted from the project portfolio PDF
- `assets/docs/resume.pdf` - resume download
- `assets/docs/project-portfolio.pdf` - original project portfolio PDF

## GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set the source to the main branch and root folder.
4. Save, then use the generated GitHub Pages URL.
